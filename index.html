What is Sunset Hills?

Sunset Hills is a coding exercise focusing on the use of arrays. An array, in JavaScript, is a collection of data that can be manipulated as one, which is especially useful when dealing with large amounts of data. In this challenge, a program is written to first read the user's input data, then collect it into an array. Once collected, a JavaScript function is then written with the purpose of determining whether or not the number given is higher than those to the left of it. If it is, then the sunset is visible to that entry. To learn more about JavaScript arrays, visit www.w3schools.com.
